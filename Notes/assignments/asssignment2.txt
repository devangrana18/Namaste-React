Q3 What is parcel-cache

    parcel-cache is a directory created by Parcel bundler during the build process to store cached data and 
    improve build performance. When Parcel bundles your project, it performs various operations such as dependency 
    resolution, transformation, and optimization. To speed up subsequent builds, it caches the results of these 
    operations in the parcel-cache directory

Q4 What is npx ?

    npx is a command-line tool that comes bundled with Node.js and npm. 
    It is used to run Node.js packages and execute command-line tools without the need to install them globally on your system.
